# kanban
## Technologies
* Front-end: SCSS, JS
* Back-end: Node.js 15.3
* Database: MongoDB: 5.0.5

## Setup
### MongoDB
* Start MongoDB
* Create new database (kanban-db)
* Import Collections (optional)
### Project
```
npm i
node app.js
```
## Application
http://localhost:3000/
